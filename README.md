# Pandas Data Science Bootcamp 📊

This repository serves as a technical log of my transition from a **Python Automation Developer** to a **Data Scientist/AI Engineer**. It contains deep-dive analysis into large-scale datasets using the Pandas library.

## 🚀 The Objective
My goal is to master the full data pipeline: 
1. **Acquisition** (Web Scraping with Selenium)
2. **Analysis** (Pandas/NumPy) 
3. **Prediction** (Scikit-Learn)

## 📁 Project Structure
- `Pandas_Data_Analysis_Mastery.ipynb`: The main technical notebook covering:
    - DataFrames & Series Mechanics
    - Advanced Indexing (loc/iloc)
    - Complex Filtering & Boolean Masking
    - [Coming Soon] Data Cleaning & Transformation
    - [Coming Soon] Real-world application on Internshala Scraper data

## 🛠️ Tech Stack
- **Language:** Python 3.14.4 (Global Env)
- **Library:** Pandas, NumPy
- **Environment:** Jupyter Notebooks / VS Code
